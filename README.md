# LocalSend Portable

## 📦 Overview  
`LocalSend Portable` is a portable version of the open-source file-sharing application **LocalSend**, packaged for Windows so you can run it without installation. The package is published on PortableApps.com.  
It enables quick file (and folder) sharing between nearby devices without relying on cloud storage.

## 🔍 Features  
- Share files and folders easily with devices on the same local network (LAN).  
- No internet required — data stays local.  
- Free and open-source.  
- Portable: download once, run from USB or external drive, no install required.  
- Cross-platform base (LocalSend) though this variant is targeted for Windows portable usage.

## 🛠️ Getting Started  
1. Download the `LocalSend Portable` package from PortableApps.com (see link below).  
2. Extract (or run) the portable `.exe` from your USB or any folder.  
3. Launch the application and follow the on-screen instructions to share/receive files.  
4. On other devices in the same network, open the LocalSend client (or browser based/share link) and join the session.

## 💡 Why Use the Portable Version?  
- **Install-free**: No registry entries, no installer required.  
- **Mobility**: Carry it on a USB stick, use it on different machines (Windows) without impacting system.  
- **Local-network sharing**: Ideal for file transfers in environments with limited or no internet access.

## 🧭 Usage Tips  
- Make sure all devices (sender & receiver) are connected to the **same local network (WiFi or LAN)**.  
- If firewalls/antivirus block, allow the app to access network & communicate.  
- For large files, connect via wired LAN (if available) for faster speeds.  
- Always verify the SHA-256 when downloading from third-party mirror to ensure integrity.
